## 📦 제공되는 파일

### 📋 문서 파일 (읽어야 할 것)
- **`QUICKSTART.md`** ← **여기서 시작하세요!** (5분 안에 시작)
- **`README.md`** - 상세 설명서 및 가이드

- **`portal.py`** - 포탈 메인 애플리케이션 (10KB)
- **`tool_meta.json`** - 도구 메타데이터 (자동 수정됨)

### 🛠️ 도구 파일 (당신의 도구들)

- `tools/comparator.py` - 파일 비교도구

### 🐳 배포 파일
- **`deploy.sh`** - 자동 배포 스크립트 (한 줄 실행!)
- **`docker-compose.yml`** - Docker 실행 설정
- **`Dockerfile`** - Docker 이미지 정의
- **`requirements.txt`** - Python 의존성
- **`.streamlit/config.toml`** - Streamlit 설정

## 🚀 시작하는 3가지 방법

### 방법 1️⃣: 가장 쉬운 방법 (Docker 자동 배포)

```bash
cd amp-tools-portal
chmod +x deploy.sh
./deploy.sh
```

✅ **1-2분 후** `http://localhost:8501` 에서 포탈 실행

### 방법 2️⃣: Docker Compose 직접 실행

```bash
cd amp-tools-portal
docker-compose up -d
```

✅ **30초 후** 포탈 실행

### 방법 3️⃣: 로컬 개발 환경

```bash
cd amp-tools-portal
pip install -r requirements.txt
streamlit run portal.py
```

✅ **즉시** 포탈 실행

## ❓ 자주 묻는 질문

### Q: 포탈을 어디서 실행하나요?
**A:** 어디서나 가능합니다.
- 로컬 PC: `streamlit run portal.py`
- Docker: `docker-compose up -d`
- 서버: `deploy.sh` 실행

### Q: 누가 접근할 수 있나요?
**A:** 설정에 따라 다릅니다.
- 로컬만: `localhost:8501`
- 같은 네트워크: `your-ip:8501`
- 인터넷: 방화벽 설정 + 도메인 필요

### Q: 도구를 추가할 수 있나요?
**A:** 예! 포탈에서 직접 추가 가능합니다.
1. "⚙️ 관리자 설정" 펼치기
2. .py 파일 업로드
3. 정보 입력하고 추가

### Q: 설명을 수정할 수 있나요?
**A:** 예! 포탈에서 직접 수정 가능합니다.
1. "⚙️ 관리자 설정" → "도구 설명 수정"
2. 도구 선택 후 설명 입력
3. "💾 저장"

### Q: 24/7 운영하려면?
**A:** Docker 배포 추천합니다.
```bash
docker-compose up -d  # 백그라운드 실행
# 서버 재부팅 후에도 자동 시작
```

### Q: 외부 인터넷에서 접근하려면?
**A:** 추가 설정 필요합니다.
- nginx/Apache로 리버스 프록시 설정
- SSL 인증서 설정 (Let's Encrypt)
- 방화벽 포트 개방

## 📞 문제가 생기면

### 1️⃣ 포탈이 안 열려요
```bash
# Docker 상태 확인
docker-compose ps

# 로그 보기
docker-compose logs -f

# 재시작
docker-compose restart
```

### 2️⃣ 포트 8501이 이미 사용 중
```bash
# docker-compose.yml 수정
ports:
  - "8080:8501"  # 8501 대신 8080 사용
```

### 3️⃣ 도구가 실행 안 돼요
```bash
# main() 함수 확인
# 필요한 라이브러리 설치 확인
# 파일명 및 경로 확인
```

### 4️⃣ 더 자세한 도움이 필요하면
- `README.md` - 상세 가이드
- `ARCHITECTURE.md` - 기술 구조
